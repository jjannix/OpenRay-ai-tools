# OpenRouter AI Tools for Raycast
<img src="assets/extension-icon.png" alt="Extension Icon" width="128" height="128"/>

This extension provides AI-powered tools to interact with models on OpenRouter via Raycast.

## Features

- **Proofreader**: Automatically corrects grammar, spelling, and style errors in your text, or changes the style of your text to a more professional or casual tone.
- **Translate**: Translates text between multiple languages

## Setup

1. **Get an OpenRouter API Key**:
   - At [OpenRouter](https://openrouter.ai/)
   - Generate an API key from your account dashboard

2. **Configure the Extension**:
   - Open Raycast preferences or run a command from the extension.
   - Navigate to the OpenRouter Assistant extension
   - Enter your API key in the "OpenRouter API Key" field
   - Enable "Nerd Stats" to see additional information about the AI's response

## Agents

The extension currently provides the following AI agents:

- **Proofreader**: Corrects grammar, spelling, and style while preserving the original meaning and language.
- **Translate**: Translates text between multiple languages.

## Usage

### Proofreader

1. Copy the text you want to proofread to your clipboard
2. Open Raycast and search for "Proofread"
3. The extension will automatically process your text and display corrections
4. Use the action panel to copy the corrected text back to clipboard or change the style of your text.

### Translate

1. Copy the text you want to translate to your clipboard
2. Open Raycast and search for "Translate"
3. Choose the target language from the menu
4. The extension will automatically process your text and display the translation
5. Use the action panel to copy the translated text back to clipboard
